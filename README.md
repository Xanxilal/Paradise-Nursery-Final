# Paradise Nursery - E-Commerce Plant Shop

Welcome to **Paradise Nursery**, your one-stop destination for beautiful and healthy houseplants!

## Project Overview

Paradise Nursery is a modern e-commerce platform dedicated to bringing nature into your home. We offer a curated selection of houseplants across multiple categories, each carefully sourced and maintained to ensure quality and longevity.

## Features

- **Browse Our Collection**: Explore a wide variety of houseplants organized by category
- **Easy Shopping**: Add plants to your cart with a single click
- **Shopping Cart**: Manage your items, adjust quantities, and view total costs
- **Responsive Design**: Seamless experience across all devices
- **Redux State Management**: Efficient cart management with Redux

## Categories

- Succulents
- Flowering Plants
- Foliage Plants
- Cacti
- Herbs
- Air Plants

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm start`
4. Visit `http://localhost:3000`

## Technologies Used

- React.js
- Redux Toolkit
- CSS3
- React Router
